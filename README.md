# What is this ?

Make your text editing easier, so you can focus on building the program

# Installation

`npm i texttyer --save`

# What the feature ?

(1.0.0 => March 10 2021)

<h2>Change case formatter</h2>

<ol>
<li>Sentence case</li>
<li>lowercase</li>
<li>UPPERCASE</li>
<li>Capitalize Each Word</li>
<li>tOOGLE cASE</li>
</ol>

# How to use it

```
import { texttyer } as tt from 'texttyer';

tt.upperCase(something)

```
